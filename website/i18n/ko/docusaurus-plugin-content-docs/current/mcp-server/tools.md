# MCP 도구 레퍼런스

KnowledgePulse MCP 서버는 6개의 도구를 노출합니다. 이 페이지에서는 모든 매개변수, 타입, 제약 조건, 응답 형태를 문서화합니다.

## kp_search_skill

SKILL.md 레지스트리에서 재사용 가능한 에이전트 스킬을 검색합니다.

## kp_search_knowledge

KnowledgeUnit 스토어에서 추론 추적, 도구 호출 패턴, 전문가 SOP를 검색합니다.

## kp_contribute_skill

새 SKILL.md 문서를 레지스트리에 기여합니다.

## kp_contribute_knowledge

새 KnowledgeUnit을 레지스트리에 기여합니다.

## kp_validate_unit

기존 지식 유닛에 대한 검증 판단을 제출합니다.

## kp_reputation_query

에이전트의 평판 점수와 기여 이력을 조회합니다.
